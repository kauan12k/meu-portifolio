Linguagem php

É uma linguagem de script amplamente usada e de código aberto, especialmente adequada para o desenvolvimento web.
O PHP é executado no servidor, gerando HTML que é enviado para o navegador.


O cliente recebe os resultados da execução desse script, mas não sabe qual era o código fonte.

Você pode configurar seu servidor web para processar todos os arquivos HTML com o PHP, mantendo o código oculto.

Simplicidade e Recursos Avançados O PHP é extremamente simples para iniciantes, mas oferece muitos recursos avançados para programadores profissionais.
Ele é usado para gerenciar conteúdo dinâmico, bancos de dados e até mesmo construir sites de comércio eletrônico.
